`src/db` 디렉토리에 `index.ts` 파일을 생성하고 데이터베이스 연결을 초기화해 보겠습니다.

```typescript
import { drizzle } from 'drizzle-orm/vercel-postgres';

const db = drizzle();
```

기존 드라이버를 제공해야 한다면 다음과 같이 작성합니다.

```typescript
import { sql } from '@vercel/postgres';
import { drizzle } from 'drizzle-orm/vercel-postgres';

const db = drizzle({ client: sql });
```
