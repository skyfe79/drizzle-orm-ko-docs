## 개선 사항

LibSQL 마이그레이션이 트랜잭션 대신 배치 실행을 사용하도록 업데이트되었습니다. [문서](https://docs.turso.tech/sdk/ts/reference#batch-transactions)에 명시된 대로, LibSQL은 이제 배치 작업을 지원합니다.

> 배치는 암시적 트랜잭션 내에서 순차적으로 실행되는 여러 SQL 문으로 구성됩니다. 백엔드가 트랜잭션을 처리하며, 성공 시 모든 변경 사항이 커밋되고, 실패 시 전체 롤백이 발생하여 아무런 수정도 이루어지지 않습니다.

Turso와 Drizzle을 시작하려면 [문서](/docs/get-started-sqlite#turso)를 참고하세요.
