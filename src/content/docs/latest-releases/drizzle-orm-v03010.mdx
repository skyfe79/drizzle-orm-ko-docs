## 새로운 기능

### 🎉 `.if()` function added to all WHERE expressions

#### 조회수가 100 이상인 모든 게시물 선택하기

```ts {5}
async function someFunction(views = 0) {
  await db
    .select()
    .from(posts)
    .where(gt(posts.views, views).if(views > 100));
}
```

## 버그 수정

- AWS DataAPI의 세션 `.all`, `.values`, `.execute` 함수에 대한 내부 매핑 문제를 수정했습니다.

AWS DataAPI 시작 가이드는 [문서](/docs/get-started-postgresql#aws-data-api)에서 확인할 수 있습니다.
