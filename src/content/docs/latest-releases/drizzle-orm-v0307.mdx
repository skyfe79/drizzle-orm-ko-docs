## 수정 사항

- `@vercel/postgres` 패키지에 대한 매핑 추가

Vercel Postgres에 대해 더 알아보려면 [여기](https://vercel.com/docs/storage/vercel-postgres)를 참고하세요. Drizzle와 Vercel Postgres를 시작하려면 [문서](/docs/get-started-postgresql#vercel-postgres)를 확인하세요.

- `neon` 드라이버의 간격 매핑 수정 - [#1542](https://github.com/drizzle-team/drizzle-orm/issues/1542)

Neon에 대해 더 알아보려면 [여기](https://neon.tech/)를 참고하세요. Drizzle와 Neon을 시작하려면 [문서](/docs/get-started-postgresql#neon)를 확인하세요.
