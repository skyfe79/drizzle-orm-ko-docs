## 새로운 기능

- Neon HTTP 드라이버의 배치 API에 raw 쿼리 지원 추가 (`db.execute(...)`)

Neon과 Drizzle를 시작하려면 [문서](/docs/get-started-postgresql#neon)를 참고하세요.


## 수정 사항

- `@neondatabase/serverless` HTTP 드라이버 타입 문제 해결 ([#1945](https://github.com/drizzle-team/drizzle-orm/issues/1945), [neondatabase/serverless#66](https://github.com/neondatabase/serverless/issues/66))
- SQLite 프록시 드라이버 `.run()` 결과 수정

SQLite 프록시 드라이버와 Drizzle을 시작하려면 [문서](/docs/get-started-sqlite#http-proxy)를 참고하세요.
