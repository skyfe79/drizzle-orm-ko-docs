## 커뮤니티 기여 릴리스 🎉

### 내부 기능 및 변경 사항

- d1에 대한 테스트 세트 추가
- 내부 문서 문제 수정


### 수정 사항

- MySQL에서 타임스탬프 밀리초가 잘리는 문제 해결
- SQLite 기반 방언(dialects)에서 `.get()` 메서드의 타입 수정 ([#565](https://github.com/drizzle-team/drizzle-orm/issues/565))
- 쿼리가 두 번 실행되던 SQLite 프록시 버그 수정


### 새로운 패키지 🎉

[drizzle-typebox](/docs/typebox) 패키지에 [Typebox](https://github.com/sinclairzx81/typebox) 지원이 추가되었습니다.

더 많은 사용 예제는 문서 페이지를 확인해 주세요: /docs/typebox
