## 새로운 기능

### 🎉 xata-http 드라이버 지원

**[공식 웹사이트](https://xata.io)**에 따르면, Xata는 신뢰성, 확장성, 그리고 개발자 경험에 초점을 맞춘 Postgres 데이터 플랫폼입니다. Xata Postgres 서비스는 현재 베타 버전이며, 계정에서 이를 활성화하는 방법은 [Xata 문서](https://xata.io/docs/postgres)를 참고하세요.

Drizzle ORM은 Xata Postgres 데이터베이스에 접근하기 위해 `drizzle-orm/xata` 패키지와 함께 제공되는 `xata` 드라이버와 **`postgres`** 또는 **`pg`** 드라이버를 기본적으로 지원합니다.

아래 예제는 [xata init](https://xata.io/docs/getting-started/installation) CLI 커맨드를 실행하여 생성된 Xata 클라이언트를 사용합니다.

<Npm>
drizzle-orm @xata.io/client
-D drizzle-kit
</Npm>

```ts
import { drizzle } from 'drizzle-orm/xata-http';
import { getXataClient } from './xata'; // 생성된 클라이언트

const xata = getXataClient();
const db = drizzle(xata);

const result = await db.select().from(...);
```

`pg` 또는 `postgres.js` 드라이버를 사용하여 Xata에 연결할 수도 있습니다.

Xata와 Drizzle을 시작하려면 [문서](/docs/get-started-postgresql#xata)를 참고하세요.
