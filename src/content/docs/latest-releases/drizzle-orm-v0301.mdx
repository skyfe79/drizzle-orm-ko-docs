## 새로운 기능

### 🎉 OP-SQLite 드라이버 지원

사용 예제

```ts copy {4,5,6,8}
import { open } from '@op-engineering/op-sqlite';
import { drizzle } from 'drizzle-orm/op-sqlite';

const opsqlite = open({
	name: 'myDB',
});

const db = drizzle(opsqlite);

await db.select().from(users);
```

더 많은 사용법과 설정 정보는 [op-sqlite 문서](http://orm.drizzle.team/docs/get-started-sqlite#op-sqlite)를 참고하세요.
