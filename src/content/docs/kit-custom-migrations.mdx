# Drizzle Kit으로 마이그레이션하기

<Prerequisites>
- Drizzle과 `drizzle-kit` 시작하기 - [여기서 읽기](/docs/get-started)
- Drizzle 스키마 기본 개념 - [여기서 읽기](/docs/sql-schema-declaration)
- 데이터베이스 연결 기본 사항 - [여기서 읽기](/docs/connect-overview)
- Drizzle 마이그레이션 기본 개념 - [여기서 읽기](/docs/migrations)
- Drizzle Kit [개요](/docs/kit-overview) 및 [설정 파일](/docs/drizzle-config-file)
- `drizzle-kit generate` 커맨드 - [여기서 읽기](/docs/drizzle-kit-generate)
- `drizzle-kit migrate` 커맨드 - [여기서 읽기](/docs/drizzle-kit-migrate)
</Prerequisites>

Drizzle는 여러분이 직접 커스텀 SQL 마이그레이션 파일을 작성할 수 있도록 빈 마이그레이션 파일을 생성할 수 있게 해줍니다. 이는 현재 Drizzle Kit에서 지원하지 않는 DDL 변경이나 데이터 시딩을 위해 사용할 수 있습니다. 이후 [`drizzle-kit migrate`](/docs/drizzle-kit-migrate) 커맨드로 이를 실행할 수 있습니다.

```shell
drizzle-kit generate --custom --name=seed-users
```
