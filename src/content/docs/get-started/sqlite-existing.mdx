<Callout type='info' title='중요'>
예를 들어, 테스트 목적으로 프로젝트 루트에 SQLite 데이터베이스 파일을 생성하려면, `LibSQL`에서 요구하는 형식인 `file:`을 실제 파일명 앞에 붙여야 합니다. 다음과 같이 작성하면 됩니다:
```plaintext copy
DB_FILE_NAME=file:local.db
```
더 자세한 정보는 **[LibSQL 문서](https://docs.turso.tech/sdk/ts/reference#local-development)**를 참고하세요.
</Callout>

#### 9단계 - 데이터베이스에 변경 사항 적용하기 (선택 사항)

<ApplyChanges/>


위 예제에서는 `applyDatabaseChanges` 함수를 사용하여 데이터베이스에 변경 사항을 적용합니다. 각 변경 사항은 배열로 정의되어 있으며, 함수 내에서 각 변경 사항을 처리합니다. 이 예제는 데이터베이스 변경 사항을 적용하는 기본적인 방법을 보여줍니다. 실제 프로젝트에서는 데이터베이스 스키마 변경, 데이터 마이그레이션 등 다양한 작업이 포함될 수 있습니다.
