# Drizzle와 Bun:SQLite 시작하기

<Prerequisites>  
  - **bun** - 자바스크립트 올인원 툴킷 - [여기서 읽기](https://bun.sh/)
  - **bun:sqlite** - 고성능 SQLite3 드라이버의 네이티브 구현 - [여기서 읽기](https://bun.sh/docs/api/sqlite)
</Prerequisites>

<Callout type='info' title='중요'>

```plaintext copy
DB_FILE_NAME=mydb.sqlite
```
