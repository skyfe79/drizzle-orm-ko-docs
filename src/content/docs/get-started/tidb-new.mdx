# Drizzle와 TiDB 시작하기

<Prerequisites>  
  - **dotenv** - 환경 변수를 관리하기 위한 패키지 - [여기서 읽기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일을 실행하기 위한 패키지 - [여기서 읽기](https://tsx.is/)
  - **TiDB** - PingCAP의 분산 SQL 데이터베이스 - [여기서 읽기](https://www.pingcap.com/)
  - **serverless-js** - HTTP 외부 연결이 필요한 서버리스 및 엣지 컴퓨팅 플랫폼을 위한 패키지 - [여기서 읽기](https://github.com/tidbcloud/serverless-js)
</Prerequisites>

<Callout title='중요' type='warning'>
이 튜토리얼에서는 **HTTP** 호출을 위해 `@tidbcloud/serverless` 드라이버를 사용합니다. TCP를 통해 TiDB에 연결해야 한다면 [MySQL 시작하기](/docs/get-started/mysql-new) 페이지를 참고하세요.
</Callout>
