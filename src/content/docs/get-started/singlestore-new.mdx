# Drizzle와 SingleStore 시작하기

<Prerequisites>  
  - **dotenv** - 환경 변수를 관리하기 위한 패키지 - [여기서 읽기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일을 실행하기 위한 패키지 - [여기서 읽기](https://tsx.is/)
  - **mysql2** - MySQL 데이터베이스를 쿼리하기 위한 패키지 - [여기서 읽기](https://github.com/sidorares/node-mysql2)
</Prerequisites>

Drizzle를 SingleStore 데이터베이스와 함께 사용하려면 `singlestore` 드라이버를 사용해야 합니다.

**[공식 웹사이트](https://github.com/sidorares/node-mysql2)**에 따르면, `mysql2`는 성능에 초점을 맞춘 Node.js용 MySQL 클라이언트입니다.

Drizzle ORM은 SingleStore 데이터베이스를 위해 `drizzle-orm/singlestore` 패키지를 통해 `mysql2`를 기본적으로 지원합니다.
