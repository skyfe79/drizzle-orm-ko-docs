# Drizzle와 MySQL 시작하기

<Prerequisites>  
  - **dotenv** - 환경 변수를 관리하기 위한 패키지 - [여기서 읽기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일을 실행하기 위한 패키지 - [여기서 읽기](https://tsx.is/)
  - **mysql2** - MySQL 데이터베이스를 쿼리하기 위한 패키지 - [여기서 읽기](https://github.com/sidorares/node-mysql2)
</Prerequisites>

Drizzle를 MySQL 데이터베이스와 함께 사용하려면 `mysql2` 드라이버를 사용해야 합니다.

**[공식 웹사이트](https://github.com/sidorares/node-mysql2)**에 따르면, `mysql2`는 성능에 초점을 맞춘 Node.js용 MySQL 클라이언트입니다.

Drizzle ORM은 `drizzle-orm/mysql2` 패키지를 통해 `mysql2`를 기본적으로 지원합니다.

<InstallPackages lib='mysql2'/>

<Callout title='팁'>
아직 MySQL 데이터베이스가 없고 테스트용으로 하나를 만들고 싶다면, Docker에서 MySQL을 설정하는 방법에 대한 가이드를 참고하세요.

Docker에서 MySQL 설정 가이드는 [여기](/docs/guides/mysql-local-setup)에서 확인할 수 있습니다. 가이드를 따라 설정한 후, 데이터베이스 URL을 생성하고(가이드에 설명되어 있음) 다음 단계를 진행하세요.
</Callout>
