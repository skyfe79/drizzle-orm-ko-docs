# Drizzle와 SQLite 시작하기

<Prerequisites>  
  - **dotenv** - 환경 변수를 관리하기 위한 패키지 - [자세히 보기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일을 실행하기 위한 패키지 - [자세히 보기](https://tsx.is/)
  - **libsql** - 전역 애플리케이션에 적합하도록 낮은 쿼리 지연 시간을 위해 최적화된 SQLite 포크 - [자세히 보기](https://docs.turso.tech/libsql)
</Prerequisites>

Drizzle는 `libsql`과 `better-sqlite3` 드라이버를 통해 SQLite 연결을 기본적으로 지원합니다.

이 시작 가이드에서는 `libsql`을 사용할 예정입니다. SQLite에 연결하는 다른 방법을 찾고 있다면 [SQLite 연결](/docs/get-started-sqlite) 페이지를 확인하세요.

<Callout type='info' title='중요'>
예를 들어, 테스트 목적으로 프로젝트 루트에 SQLite 데이터베이스 파일을 생성하려면, `LibSQL`에서 요구하는 형식인 `file:`을 실제 파일명 앞에 붙여야 합니다. 아래와 같이 작성하면 됩니다:
```plaintext copy
DB_FILE_NAME=file:local.db
```
더 자세한 정보는 **[LibSQL 문서](https://docs.turso.tech/sdk/ts/reference#local-development)**를 참고하세요.
</Callout>
