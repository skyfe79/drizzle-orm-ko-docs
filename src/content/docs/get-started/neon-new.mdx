# Drizzle와 Neon 시작하기

<Prerequisites>
  - **dotenv** - 환경 변수를 관리하기 위한 패키지 - [자세히 보기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일을 실행하기 위한 패키지 - [자세히 보기](https://tsx.is/)
  - **Neon** - 서버리스 Postgres 플랫폼 - [자세히 보기](https://neon.tech/docs/introduction)
</Prerequisites>

Drizzle는 `neon-http`와 `neon-websockets` 드라이버를 통해 Neon 연결을 기본적으로 지원합니다. 이 드라이버들은 내부적으로 **neon-serverless** 드라이버를 사용합니다.

`neon-http`와 `neon-websockets` 드라이버를 사용하면 서버리스 환경에서 TCP 대신 HTTP 또는 WebSockets를 통해 Neon 데이터베이스에 접근할 수 있습니다. 단일 비대화형 트랜잭션의 경우 HTTP를 통한 쿼리가 더 빠릅니다.

세션 또는 대화형 트랜잭션 지원이 필요하거나 `pg` 드라이버와 완전히 호환되는 대체 드라이버가 필요하다면, WebSocket 기반의 `neon-serverless` 드라이버를 사용할 수 있습니다. [Postgres](/docs/get-started/postgresql-new)를 통해 Neon 데이터베이스에 직접 연결할 수도 있습니다.
